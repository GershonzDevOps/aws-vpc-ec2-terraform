# AWS VPC + EC2 Terraform Project

## Overview
This project provisions AWS infrastructure using Terraform.

## Technologies Used
- AWS
- Terraform
- EC2
- VPC
- Security Groups
- Internet Gateway

## Infrastructure Created
- Custom VPC
- Public Subnet
- Route Table
- Internet Gateway
- Security Group
- EC2 Instance
- Apache Web Server

## Deployment
terraform init
terraform plan
terraform apply

## Output
The EC2 instance hosts a web page deployed automatically using user_data.
